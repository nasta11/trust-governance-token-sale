# Threat Model (MVP)

## Threats
1) Admin abuse / rug pull
2) Unlimited mint path
3) Whale domination (no caps)
4) Bot spam / tx floods
5) Misconfigured parameters
6) Reentrancy / unsafe transfers
7) No transparency (no logs)

## Mitigations
- pause switch
- max per wallet
- max total sold
- strict event logging
- minimal payment flow
